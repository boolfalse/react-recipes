
## React Recipes Blog app



#### About:

**React Recipes** Blog app is a simple blog app built with React.js.


#### Installation:

- Clone the repository and install dependencies:
```bash
git clone git@github.com:boolfalse/react-recipes.git
cd react-recipes
npm install
```

- Run the app:
```bash
- Run the application:
```bash
# for development
npm run start
# for production
npm run build
```
